# 📈 **Stock Price Prediction Dashboard** ✨

A **production-ready Streamlit app** that forecasts **stock Open & Close prices** using **Deep Learning** + **live yfinance data**.

***

## 🚀 **One-Click Demo**

```
AAPL → Predict → ✅ $228.76 (+1.5%) in 1 minute!
```

## ⚡ **Complete Setup (2 Commands)**

```bash
# Install everything
pip install streamlit tensorflow yfinance plotly pandas scikit-learn numpy

# Run instantly
streamlit run stock_price_prediction.py
```

**🎉 Opens:** `http://localhost:8501`
